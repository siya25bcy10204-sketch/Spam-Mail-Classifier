Project Title:-
Spam Mail Classifier 
## Problem Statement
Spam messages can disturb users and can be harmful. This project automatically classifies SMS messages as spam if they cause harm

## Scope of the Project
- Input: SMS text
- Output: Spam
- Machine Learning model with preprocessing and prediction

## Target Users
- Students
- General Users
- Developers
- others

## High-level Features
-detect sms a spam or not
- Preprocessing
- TF-IDF vectorizer
- Naive Bayes model
- Prediction API
- Evaluation metrics
