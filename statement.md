Project Title
Spam Mail Classifier (Spam vs Ham Detection)

## Problem Statement
Spam messages disturb users and can be harmful. This project automatically classifies SMS messages as spam or ham.

## Scope
- Input: SMS text
- Output: Spam/Ham
- Machine Learning model with preprocessing and prediction

## Target Users
- Students
- General Users
- Developers

## High-level Features
- Preprocessing
- TF-IDF vectorizer
- Naive Bayes model
- Prediction API
- Evaluation metrics
